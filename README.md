# Mensabot
Mensabot TU Berlin Quality and Usability Lab Project

## Deployment & Setup

Deploying Mensabot is fully automated and interactive. To install Docker, clone this repository, and start the graphical terminal setup wizard, simply run this command:

```bash
curl -sSL https://raw.githubusercontent.com/tobiasv1337/Mensabot/main/install.sh | bash
```

> **Note:** For detailed instructions on the configuration modes, branch updating, and SSL certificates, please see the [SETUP_README.md](SETUP_README.md) guide.
